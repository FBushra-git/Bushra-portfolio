# 🚀 Professional Developer Portfolio — Bushra Tasmin

A high-performance, SEO-optimized developer portfolio designed for modern frontend and future full stack development. Built with a strong focus on smooth animations, responsive design, interactive UI effects, and scalable architecture.

---

## ✨ Key Features

### 🎨 Advanced UI/UX

* Dark & Light mode support
* Custom Nuclear Tech Cursor
* Interactive Water Wave Background using HTML5 Canvas
* Smooth transitions and animations powered by Framer Motion

### 📊 Dynamic Project Showcase

* Real-time project filtering system
* Category-based filtering for:

  * Frontend
  * Fullstack
  * AI/ML
  * Blockchain
  * Mobile
* Efficient React state management

### 🔍 SEO & Metadata Optimization

* OpenGraph metadata support
* JSON-LD Schema.org structured data
* Dynamic sitemap generation
* Optimized page structure for better search engine visibility

### 📱 Fully Responsive Design

* Mobile-first development approach
* Optimized for:

  * Smartphones
  * Tablets
  * Laptops
  * 4K Displays

### ⚙️ Data-Driven Architecture

* Easily manageable content system
* Portfolio data powered by:

  * `site.json`
  * `projects.json`
* No need to modify component logic for content updates

| Layer            | Technologies                           |
| ---------------- | -------------------------------------- |
| Frontend         | React 18, Vite 5, Tailwind CSS         |
| Animation        | Framer Motion, HTML5 Canvas            |

| SEO              | React Helmet, JSON-LD, OpenGraph       |
| State Management | React Hooks, Context API               |
